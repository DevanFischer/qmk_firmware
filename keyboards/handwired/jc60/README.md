JC60 Keyboard
=============

A design by Jeremy Cowgar <jeremy@cowgar.com>.

Wiring
------

The JC60 is setup as a simple 5x13 grid using a Teensy 2.0 as the controller.

| Pin | Purpose |
|-----|---------|
| B0  | Row 1   |
| B1  | Row 2   |
| B2  | Row 3   |
| B3  | Row 4   |
| B7  | Row 5   |
| D0  | Col 1   |
| D1  | Col 2   |
| D2  | Col 3   |
| D3  | Col 4   |
| C6  | Col 5   |
| C7  | Col 6   |
| D5  | Col 7   |
| D4  | Col 8   |
| D6  | Col 9   |
| D7  | Col 10  |
| B4  | RGB     |
| B5  | Col 11  |
| B6  | Col 12  |
| F7  | Col 13  |
